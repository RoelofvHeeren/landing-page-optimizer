# Project Context: 6-Week Transformation Funnel

This repository contains the standalone funnel for the "6 Week Transformation" program.

## 🎯 Goal
A high-impact landing page focused on video testimonials, pillars of success, and clear call-to-actions for the 6-week program.

## 🏗️ Technical Stack
- **Framework**: Vite
- **Deployment**: `npm run dev` (Port 5174)

## 🗝️ Specific Features
- **Video Heroes**: Heavy use of background and interactive video thumbnails (Testimonials).
- **Pillars Interaction**: A 5-column grid showing the transformation foundation (Training, Nutrition, etc.).
- **Offer Box**: A high-impact CTA box at the bottom of the page.

## 🛠️ Modifying this project
- Primary styles are in `style.css`. 
- When editing, ensure the premium "video-heavy" aesthetic is maintained.
