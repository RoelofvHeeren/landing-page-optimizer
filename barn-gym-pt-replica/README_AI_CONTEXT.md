# Project Context: Personal Training Landing Page

This repository is dedicated specifically to the high-converting Personal Training (PT) landing page funnel.

## 🎯 Goal
A standalone, highly optimized funnel for PT leads, featuring the interactive survey results and social proof marquee.

## 🏗️ Technical Stack
- **Framework**: Vite
- **Deployment**: `npm run dev` (Port 5175)

## 🗝️ Specific Features
- **Marquee Results**: An infinite-scroll CSS marquee for transformation photos.
- **Survey Modal**: An Apple-style survey funnel triggered by user behavior or buttons.
- **Specific Styles**: This project uses `Raleway` as the primary body font.

## 🛠️ Modifying this project
- Ensure all behavioral triggers (buttons, exit intent) correctly launch the modal defined in `index.html`.
- Asset pathing: Images are located in `/public/images/`.
